Tailwind CSS Theme Guide

A comprehensive collection of design systems with four distinct styles: Feel, Flow, Find, and Flourish.

How to Use This Guide

This guide presents four complete design systems that can be used as a foundation for your projects. Each system includes:

  • Color palette with primary, secondary, and accent colors
  • Typography styles and font pairings
  • Component examples including buttons, cards, forms, and more
  • Layout patterns and spacing guidelines

Navigate through the sections using the menu above to explore each theme in detail.
